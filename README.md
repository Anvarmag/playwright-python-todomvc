# Playwright Python E2E Tests (TodoMVC)

Demo project showcasing end-to-end test automation using Playwright (Python)
with production-ready structure and CI/CD.

## Stack

- Python 3.12
- Playwright
- pytest
- pytest-html
- GitHub Actions
- GitHub Pages

## Test Report

Latest HTML test report:
https://anvarmag.github.io/playwright-python-todomvc/

## Project structure

- pages/ - Page Object Model
- tests/ - E2E tests and fixtures
- .github/workflows/ - CI/CD pipeline

## CI/CD

- Tests run on every push to main
- HTML report is automatically deployed to GitHub Pages
